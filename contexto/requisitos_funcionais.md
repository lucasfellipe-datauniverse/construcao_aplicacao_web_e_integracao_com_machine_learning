### Requisitos funcionais do software

Abaixo estão os requisitos funcionais do Projeto 1 conforme discutido com as áreas de
negócio e resultado do levantamento de requisitos.

RF1 - Entrada de Dados: A aplicação deve permitir que o usuário insira informações
referentes a:

· País de atuação do desenvolvedor.
· Nível educacional (por exemplo, ensino médio, graduação, pós-graduação).
· Função específica (por exemplo, Desenvolvedor Front-end, Engenheiro de Machine
Learning, Cientista de Dados).
· Anos de experiência na área.

RF2 - Processamento de Previsão: Após a entrada dos dados pelo usuário, a aplicação
deve processar a previsão utilizando o modelo de Machine Learning treinado e apresentar o
salário previsto.

RF3 - Interface do Usuário: A aplicação deve oferecer uma interface amigável e intuitiva
que guie o usuário através do processo de inserção de dados e visualização da previsão de salário.

RF4 - Validação dos Dados de Entrada: A aplicação deve validar os dados inseridos pelo
usuário para garantir que estão no formato correto antes de realizar a previsão.

RF5 - Exploração dos Dados: A aplicação deve permitir que o usuário explore os dados
através de gráficos.
